# ICB Value Packs

Relates to Specialist Analytics request SAT0014 - ICB Value Packs.

## Contents

Repository for a Streamlit app to display Planner data for the "Local report usage" aspect of the value packs.

## Data

Contains administrative data extracted from BOB/Frimley Client Facing Team's Planner page.

## Requirements

requirements.txt was the original setup that I was using. Package version numbers were removed so that uv could determine which version numbers to use.

uv_requirements.txt is what uv decided on.
